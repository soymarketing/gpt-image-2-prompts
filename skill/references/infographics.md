# Infographics

Use this reference when the user wants comparison graphics, diagrams, labeled visuals, slide-ready explainers, charts, menus, posters with structured text, or any text-heavy composition.

## What matters most

Prioritize:
- exact text control
- strong hierarchy
- legibility
- predictable layout
- minimal visual noise
- alignment and spacing

## Prompt structure for infographic work

```text
Create a [infographic / comparison chart / labeled diagram / explainer slide] for [topic].
Layout: [vertical / square / horizontal / two-column / grid]
Sections: [what blocks must exist]
Main visual system: [icons, diagrams, simple illustrations, photo-based panels]
Text to include: "[EXACT TEXT]"
Typography: [bold sans-serif / technical / editorial / clean UI]
Color system: [palette]
Style: [clean, editorial, technical, modern]
Constraints: no extra text, perfect legibility, aligned sections, no clutter, strong visual hierarchy
```

## Good language for infographics

Use phrases like:
- `crisp typography`
- `editorial infographic`
- `clear section hierarchy`
- `aligned columns`
- `high contrast`
- `clean labels`
- `presentation-ready`
- `minimal visual noise`

## Common infographic patterns

### 1. Comparison chart

```text
Create a clean comparison infographic for two AI tools.
Layout: vertical infographic with two equal columns and a clear header section.
Sections: overview, speed, quality, best use case, pricing.
Main visual system: simple polished icons and clean comparison rows.
Text to include: "TOOL A", "TOOL B", "Speed", "Quality", "Best For", "Pricing".
Typography: bold clean sans-serif with strong hierarchy.
Color system: off-white background, dark charcoal text, muted blue and green accents.
Style: professional modern editorial infographic.
Constraints: no extra text, crisp typography, aligned columns, no clutter, no watermark.
```

### 2. Process diagram

```text
Create a clean process infographic showing a 4-step AI image workflow.
Layout: horizontal flow with four connected steps.
Sections: brief, prompt, generate, refine.
Main visual system: minimal icons, arrows, clean labeled boxes.
Text to include: "1. Brief", "2. Prompt", "3. Generate", "4. Refine".
Typography: modern bold sans-serif, highly legible.
Color system: white background with navy and electric blue accents.
Style: presentation-ready, minimal, modern.
Constraints: no extra text, no clutter, strong hierarchy, perfectly legible labels.
```

## Rules for text-heavy work

- Put literal text in quotes.
- If spelling matters, give the exact spelling carefully.
- Say `no extra text` almost every time.
- Ask for `high contrast` and `perfect legibility`.
- Explicitly define sections, not just the topic.

## Revision prompts for infographic work

- `Keep the structure, improve legibility and spacing.`
- `Preserve the content, reduce visual clutter and make it more editorial.`
- `Keep all labels, make the hierarchy clearer.`
- `Do not change the wording; align columns and improve balance.`
- `Make the infographic simpler and more presentation-ready.`

## Failure fixes

### Wrong or broken text
Add:
- exact quoted text
- simpler wording
- fewer text blocks per image
- `no extra text`
- `crisp typography`

### Feels messy
Add:
- fixed section list
- cleaner grid
- fewer decorative elements
- explicit alignment instructions

### Looks too playful when you need serious
Add:
- `editorial`
- `technical`
- `presentation-ready`
- `minimal visual noise`
